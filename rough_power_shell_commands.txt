Prerequisites (what to install)

Windows (10/11) — up to date.

VS Code — installed. (https://code.visualstudio.com/
)

Recommended extensions: Python, Remote - Containers (optional).

Docker Desktop — install and run Docker for Windows. Enable WSL 2 backend if prompted. (https://www.docker.com/products/docker-desktop
)

Git (optional but useful) — https://git-scm.com/

Python 3.10+ — install from https://www.python.org/
 and check python --version.

(Optional) Conda (Miniconda) — helpful for environment management: https://docs.conda.io/en/latest/miniconda.html
